# Student Support & Risk Insights

### Project Overview
This feature improves the Student Checker Project by helping instructors identify students who may need academic support.
The system analyzes student scores, assigns a risk level, provides improvement advice, and generates a risk summary for the class.

### Features
- Assigns High, Medium, and Low risk levels
- Generates support advice for each student
- Displays a complete student report
- Provides a class risk summary
- Handles invalid score input
